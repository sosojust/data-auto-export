# 项目结构
## frontend - Web管理系统的前端应用
**功能**：前端应用代码，基于Vue.js构建，负责用户界面和交互。
## api - web管理系统的后端接口
**功能**：后端接口代码，基于Flask构建，负责处理前端请求、业务逻辑和数据库交互。
## cli - 命令行和定时任务调度器
**功能**：命令行工具和定时任务调度器代码，基于Python构建，负责执行定时任务和提供命令行接口。
## core - 所有核心业务逻辑能力
**功能**：核心业务逻辑代码，包括为web系统、cli提供数据查询服务、任务管理、导出服务等各项功能代码。
